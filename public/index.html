ChatFluence

Confluence 문서를 자연어로 검색·조회하고, 필요하면 페이지·댓글 작업까지 이어갈 수 있는 AI 챗입니다. 파일을 첨부해 함께 분석할 수도 있습니다.

이용 안내

  • 상단에서 스페이스, 모델, 작업 유형(읽기만 / 페이지 생성·댓글·수정·삭제 등)을 선택한 뒤 질문을 입력하세요.
  • Confluence에서 제목·내용 검색, 특정 페이지 읽기, 스페이스 목록 등은 도구로 처리됩니다.
  • 입력창 왼쪽 클립으로 PDF·Word·슬라이드·CSV·이미지 등을 첨부하면, 본문을 읽고 질문·요약·표 정리 등에 활용합니다.
  • 첨부 내용을 바탕으로 Confluence에 요약 페이지를 만들거나 댓글을 달도록 요청할 수 있습니다. 그때는 해당 쓰기 작업을 선택해야 합니다.

유의사항

  • 페이지 생성·수정·삭제·댓글 작성은 실행 전 미리보기와 확인(실행/취소) 단계가 있을 수 있습니다. 반드시 내용을 검토한 뒤 진행하세요.
  • AI 답변·요약·추출 텍스트는 참고용입니다. 임상·법무·보안 등 중요한 판단은 원문과 정책을 기준으로 하세요.
  • 불필요한 개인정보·기밀은 채팅·첨부에 넣지 마세요. 사내 정보보안 규정을 따라 이용하세요.
  • 첨부는 개수·용량 제한이 있으며, 스캔 PDF·복잡한 서식은 텍스트가 일부만 추출되거나 빠질 수 있습니다.

문의: 이나혁 (연구개발본부 Fundus팀)

새 페이지 부모 (선택)

선택하면 다음 메시지부터 페이지 생성 시 이 부모 아래에 만듭니다. 스페이스를 바꾸면 선택이 해제됩니다. 비우면 AI가 URL·검색으로 부모를 정하거나 루트에 둘 수 있습니다.

댓글 달 페이지 (선택)

선택하면 다음 메시지부터 댓글 작성 시 이 페이지에 답니다. 스페이스를 바꾸면 선택이 해제됩니다. 비우면 AI가 URL·검색으로 대상을 정합니다.

수정할 페이지 (선택)

선택하면 페이지 수정 시 이 페이지 ID를 우선 사용합니다. 스페이스를 바꾸면 해제됩니다. 비우면 AI가 URL·검색으로 대상을 정합니다.

삭제할 페이지 (선택)

선택하면 페이지 삭제 시 이 페이지 ID를 우선 사용합니다. 스페이스를 바꾸면 해제됩니다. 비우면 AI가 URL·검색으로 대상을 정합니다.

ChatFluence는 AI Agent로 실수할 수 있습니다. 사용에 유의하시어 반드시 결과물을 확인하세요.

Settings

저장하면 서버 기본값이 바뀌고, 채팅 화면 모델 드롭다운도 이 값으로 맞춰집니다. OpenAI 키는 로그인할 때만 설정됩니다.

채팅 상단 드롭다운에 처음부터 선택돼 있을 스페이스입니다. 아래에서 드롭다운에 넣는 스페이스(글로벌 체크 + 개인 일괄 포함) 중에서만 고를 수 있습니다.
